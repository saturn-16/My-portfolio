# Gaurav Kumar - Personal Portfolio

A sleek, premium, and highly interactive 3D portfolio website showcasing full-stack development, AI engineering, and cybersecurity expertise. Inspired by modern design aesthetics (Apple, Stripe, Vercel), this website is built to feel premium, responsive, and alive.

## ✨ Key Features

- **Interactive 3D Lanyard ID Card**: A physics-backed 3D corporate ID badge built using **React Three Fiber**, **Drei**, and **Rapier Physics**. Drag it, swing it, and rotate it in real-time.
- **View Card Modal**: Click the "View Card" button to bring the ID card into full-screen view with a spring transition, blurring everything in the background for close-up readability.
- **GPU-Accelerated Preloader**: An optimized loading screen featuring a smooth signature animation and a native CSS-composited progress bar that loads instantly without main-thread stutter.
- **Tech Stack & Skill Capsules**: Clean categorization of competencies spanning Cybersecurity & Threat Intel, Full Stack Development, and AI & Machine Learning.
- **Responsive Layout**: Designed for seamless visual experiences across mobile, tablet, and desktop viewports.

## 🛠️ Technology Stack

- **Framework**: React 19 + TypeScript
- **Bundler & Build Tool**: Vite
- **Styling**: Tailwind CSS (v4)
- **3D Graphics & Physics**: Three.js, `@react-three/fiber`, `@react-three/drei`, `@react-three/rapier`
- **Animations**: `motion` (Framer Motion)
- **Smooth Scroll**: Lenis
